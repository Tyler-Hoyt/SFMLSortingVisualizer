# SFMLSortingVisualizer
Sorting visualizer using the SFML library.
This was thrown together in less than 24 hours.


## Packages Needed For SFML
1. pthread
2. opengl
3. xlib
4. udev
5. xrandr
6. xcursor
7. freetype
8. openal 
9. flac
10. vorbis

## Controls
Press space to sort.
Press r to get new data set.
